Добавляет к постам виджет "Мне нравится" из ВКонтакте.

== Description ==

Совсем недавно ВКонтакте появился функционал, позволяющий пользователям отмечать понравившиеся им материалы. Этот плагин позволяет добавить такую кнопку к каждой записи в блоге.

Плагин имеет некоторое количество настроек, перед использованием необходимо создать приложение ВКонтакте.

== Installation ==

Для работы плагина необходимо подключить свой блог на [этой странице](http://vkontakte.ru/apps.php?act=add&site=1). После создания приложения необходимо скопировать ID приложения на станице настроек (приложение можно найти [здесь](http://vkontakte.ru/apps.php)).
