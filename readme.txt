Add like button from social network VKontakte (vkontakte.ru, vk.com).
Добавляет к постам виджет "Мне нравится" из ВКонтакте

== Description ==

Совсем недавно ВКонтакте появился функционал, позволяющий пользователям отмечать понравившиеся им материалы (описание).
Из официального анонса:
При помощи виджета «Мне нравится» Вы можете за 5 минут добавить к материалам Вашего сайта соответствующую галочку. Это позволит Вашим посетителям одним нажатием кнопки мыши выразить своё отношение к статье и, что не менее важно, моментально поделиться ссылкой на статью с друзьями.
Не найдя соответствующего плагина для WordPress, я решил написать его сам, взяв за основу плагин Share on Vkontakte.

Для работы плагина необходимо подключить свой блог на этой странице. После создания приложения необходимо скопировать ID приложения на станице настроек (приложение можно найти здесь).
